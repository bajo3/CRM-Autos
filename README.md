# Autos CRM — Frontend (Next.js + Tailwind)

UI estilo dashboard (similar a demos v0). **Solo frontend + mock data**.

## Requisitos
- Node 18+ (recomendado 20)

## Instalación
```bash
npm install
npm run dev
```
Abrí: http://localhost:3000

## Rutas
- /dashboard
- /vehicles
- /leads
- /credits
- /tasks
- /settings

## Próximo paso (cuando quieras)
Conectar a Supabase:
- auth
- multi-tenant (`dealership_id`)
- RLS
- tablas: vehicles, leads, credits, tasks, users
